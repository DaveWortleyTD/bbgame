HOW IT WORKS

Left-drag paints the selected brush. Right-drag erases to floor. Alt/middle-click picks the char under the cursor. Ctrl+Z undoes.

SAVE TO GAME stores the map in this browser — the game (same server) loads it automatically, so just refresh the game tab to playtest. Saved edits survive reloads until you clear them.
